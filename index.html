Skein Woven In June 2026
DigiByte's Skein algorithm hashes the 80-byte block header with Skein512(Skein512(header)). The old code used double-SHA256 — correct for Bitcoin, wrong for DGB. pool/skein.js is a full pure-JavaScript Threefish-512 implementation: 72 rounds, 8 MIX pairs per round, key injection every 4 rounds, UBI chaining, zero native addons. A NIST known-answer test vector (Skein512(0xFF)) validates the implementation on every startup. The block submitted to the node will now actually match the header hash the Skein miners computed.

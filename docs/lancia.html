◆ What is Lancia?
◆ Purpose
One-page studio to spin up, monitor, and tear down MagoFonte mining pool instances on free cloud infrastructure. Each instance runs the full stack: Stratum server, DGB node/wallet, API, and block broadcaster.
◆ How it works
Fill in the Launch form. Lancia calls the MagoFonte API (/api/v1/lancia/instances), provisions a container on your chosen provider with all config injected as environment variables, and returns a live instance ID.
◆ Pool economics
PPLNS payouts. Pool fee fixed at 2% — funds infrastructure. Miners keep 98%. The built-in DGB wallet handles all payouts automatically once the node has synced.
◆ Prerequisites
A free account on at least one supported provider
Your provider API token (see Setup Guide below)
A DGB address to receive the 2% pool fee
❱ Provider Setup Guide ❰
Get your API token from any supported provider in minutes. Select a tab, follow the steps, then paste the token into the Launch form below.
✈
Fly.io
Recommended · Global edge · Raw TCP supported · 3 free VMs
Free tier30+ regionsDocker nativeCard required (not charged)
1
Create account
Sign up at fly.io/app/sign-up. A credit card is required for identity verification — you won't be charged on the free tier.
2
Install flyctl
Linux / macOS:
curl -L https://fly.io/install.sh | sh
Windows (PowerShell):
iwr https://fly.io/install.ps1 -useb | iex
3
Log in
flyctl auth login
Opens a browser window to authenticate.
4
Generate your token
flyctl tokens create deploy -x 999999h
Copy the token — it starts with FlyV1 fm2_ and is shown once. Or get one via the dashboard at fly.io/user/personal_access_tokens.
5
Paste into Lancia
Select Fly.io in the Provider dropdown below and paste the token into the Provider API Token field. Done.

Create account
Sign up at railway.app with GitHub or email. The Hobby plan gives $5 free compute credit per month.

Note: Render free tier only exposes HTTPS (port 443). Raw TCP Stratum requires a paid plan. Best used for the API/dashboard endpoint only — use Fly.io or Oracle for Stratum.

Oracle Cloud Free Tier
Always Free · ARM VMs · 4 OCPUs + 24 GB RAM · Best raw performance
Never expiresARM Ampere A1Full TCP supportMore setup steps
1
Create account
Sign up at oracle.com/cloud/free. Credit card required for verification — not charged. Pick your home region carefully, it can't be changed. Recommended: us-ashburn-1 or eu-frankfurt-1.
2
Create an API signing key
In the OCI Console: profile icon → User Settings → API Keys → Add API Key → Generate API Key Pair. Download the private key (.pem file) and copy the config snippet shown.
3
Paste credentials into Lancia
Select Oracle Cloud below. Paste your credentials as a JSON object:
{"user":"ocid1.user.oc1..xxx","tenancy":"ocid1.tenancy.oc1..xxx","region":"us-ashburn-1","fingerprint":"aa:bb:cc:...","privateKey":"-----BEGIN RSA PRIVATE KEY-----\n...\n-----END RSA PRIVATE KEY-----"}
4
Open Stratum port in OCI firewall
OCI Console → Networking → Virtual Cloud Networks → your VCN → Security Lists → Add Ingress Rule. Set: Source 0.0.0.0/0, Protocol TCP, Port 3333. Also open 3000 for the API.
⚙
Custom / Self-Hosted
Any server with Docker · VPS, homelab, or bare metal · Full TCP support
Any Docker hostFull controlNo vendor limits
1
Install Docker on your server
curl -fsSL https://get.docker.com | sh
Any Linux VPS or bare-metal machine works. Minimum: 1 CPU, 1 GB RAM.
2
Run the Lancia agent
docker run -d --name lancia-agent \ -p 4444:4444 \ -v /var/run/docker.sock:/var/run/docker.sock \ -e AGENT_TOKEN=your-secret-token \ ghcr.io/5mil/magofonte-lancia-agent:latest
Replace your-secret-token with any strong random string — this is what you paste into Lancia.
3
Open firewall ports
ufw allow 3333/tcp && ufw allow 3000/tcp && ufw reload
4
Paste into Lancia
Select Custom below. Enter your server IP as the agent endpoint (e.g. http://1.2.3.4:4444) and paste your token.

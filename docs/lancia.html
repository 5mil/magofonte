◆ What is Lancia?
◆ Purpose
One-page studio to spin up, monitor, and tear down MagoFonte mining pool instances on free cloud infrastructure. Each instance runs the full stack: Stratum server, DGB node/wallet, API, and block broadcaster.
◆ How it works
Paste your provider token, fill in the form, and click Launch. Lancia calls the provider's API directly from your browser — no backend required. Instance state is saved in browser storage.
◆ Pool economics
PPLNS payouts. Pool fee fixed at 2% — funds infrastructure. Miners keep 98%. The built-in DGB wallet handles all payouts automatically once the node syncs.
◆ Prerequisites
A free account on at least one supported provider
Your provider API token (see Setup Guide below)
A DGB address to receive the 2% pool fee

Recommended · Global edge · Raw TCP · 3 free VMs
Free tier30+ regionsDocker nativeCard required (not charged)
1
Create account
Sign up at fly.io/app/sign-up. A credit card is required for identity verification — you won't be charged on the free tier.
2
Get a token
Go to fly.io/user/personal_access_tokens and create a token. Or via CLI:
flyctl tokens create deploy -x 999999h

Oracle Cloud Free Tier
Always Free · ARM VMs · 4 OCPUs + 24 GB RAM · Best performance
Never expiresARM Ampere A1Full TCPMore setup steps
1
Create account
Sign up at oracle.com/cloud/free. Card required for verification — not charged. Choose your home region carefully; it can't be changed. Recommended: us-ashburn-1 or eu-frankfurt-1.
2
Create an API signing key
OCI Console → profile icon → User Settings → API Keys → Add API Key → Generate Key Pair. Download the private key and copy the config snippet.
3
Paste credentials
Select Oracle Cloud and paste your credentials as JSON:
{"user":"ocid1.user.oc1..xxx","tenancy":"ocid1.tenancy.oc1..xxx","region":"us-ashburn-1","fingerprint":"aa:bb:cc:...","privateKey":"-----BEGIN RSA PRIVATE KEY-----\n...\n-----END RSA PRIVATE KEY-----"}
4
Open firewall ports
OCI Console → Networking → VCN → Security Lists → Add Ingress Rule. Open TCP port 3333 (Stratum) and 3000 (API) from 0.0.0.0/0.
5
Use the Initialization script
When creating your instance in the OCI Console, expand Advanced Options → Initialization script and paste the script generated in the Launch form below. It installs Docker and starts MagoFonte automatically on first boot.

Custom / Self-Hosted
Any Docker host · VPS, homelab, or bare metal · Full TCP
Any Docker hostFull controlNo vendor limits
1
Install Docker
curl -fsSL https://get.docker.com | sh
Minimum: 1 CPU, 1 GB RAM.
2
Run the Lancia agent
docker run -d --name lancia-agent \ -p 4444:4444 \ -v /var/run/docker.sock:/var/run/docker.sock \ -e AGENT_TOKEN=your-secret-token \ ghcr.io/5mil/magofonte-lancia-agent:latest
Replace your-secret-token with any strong random string.
3
Open firewall ports
ufw allow 3333/tcp && ufw allow 3000/tcp && ufw reload
4
Paste into Lancia
Select Custom, enter your agent endpoint (e.g. http://1.2.3.4:4444), and paste your token.
